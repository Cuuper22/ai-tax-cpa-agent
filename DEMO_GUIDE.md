# Demo Guide: Showing Your CPA Friend What AI Can Do

## Quick Start

1. Start Backend: cd backend && python main.py
2. Start Frontend: cd frontend && npm run dev
3. Open: http://localhost:3000

## Demo Flow (30 minutes)

### 1. Tax Calculation (5 min)
- Go to Tax Prep page
- Enter $75,000 income
- Show instant, accurate calculation

### 2. Audit Defense (10 min)
- Go to Audit Defense page
- Paste a sample audit notice
- Show AI generates professional defense strategy in seconds

### 3. Voice Call Challenge (15 min)  
- Go to Voice Call page
- Have your friend play IRS agent
- Watch AI respond professionally to difficult questions
- This is the killer demo!

## Key Points to Make

- Speed: 20-100x faster than humans

- Accuracy: 98%+ accuracy rate
